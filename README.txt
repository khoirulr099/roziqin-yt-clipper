Roziqin Youtube Auto Clipper - Panduan Penggunaan

=== File yang Tersedia ===

| File                | Fungsi                                      | Cara Jalankan                     |
|---------------------|---------------------------------------------|-----------------------------------|
| start_all.bat       | Jalankan Telegram Bot + Web Streamlit       | Ketik `start_all` di terminal     |
| stop_all.bat        | Matikan semua proses                        | Ketik `stop_all` di terminal      |
| Start_All.vbs       | Jalankan Telegram + Web (double click)      | Double click file                 |
| Stop_All.vbs        | Matikan semua proses (double click)         | Double click file                 |

=== Cara Penggunaan (Paling Mudah) ===

1. Jalankan Semua (Telegram + Web)
   - Buka folder youtube-auto-clip
   - Double click Start_All.vbs
   - Atau di terminal ketik: start_all

2. Matikan Semua Proses
   - Double click Stop_All.vbs
   - Atau di terminal ketik: stop_all

3. Jalankan Manual (kalau mau pisah)
   - Telegram Bot: python3 bot.py
   - Web Streamlit: python3 -m streamlit run app.py

=== Catatan ===

- Pastikan file .env sudah berisi token dan Chat ID kamu.
- Bot harus jalan dulu sebelum orang lain bisa pakai web.
- Kalau mau stop, gunakan Stop_All.vbs atau stop_all.bat.
